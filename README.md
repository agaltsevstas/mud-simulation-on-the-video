# Аннотация
GUI приложение для симуляции грязи на видео

## Поддерживаемые платформы
* Linux 

## Требования:
* python3, библиотеки:
  * sys
  * os
  * nummpy
  * cv2
  * tkinter

### Запуск
```
python3 mud_simulation_on_the_video.py
```

# Описание
Запуск осуществляется командой
```
python3 mud_simulation_on_the_video.py
```
